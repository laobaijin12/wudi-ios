# Wudi 

一款基于云机/容器的即时通讯 iOS 客户端，支持多实例会话、消息收发、媒体预览、话术/图片模板等能力。

---

## 项目结构

```
wudi-ws/
├── ios_app/                 # iOS 工程
│   ├── WudiApp/             # 主应用（SwiftUI）
│   │   ├── AppState.swift   # 全局状态
│   │   ├── Account/         # 账号·云机管理
│   │   ├── Chat/            # 聊天列表、会话、消息
│   │   ├── Main/            # Tab 主框架
│   │   ├── Network/         # 网络与缓存
│   │   ├── Services/        # 业务接口（账号、聊天、鉴权）
│   │   └── Tools/           # 工具页（话术/图片模板）
│   ├── README.md            # 媒体流接口说明（缩略图/原图）
│   └── WudiApp.xcodeproj
└── REDEME                   # 本说明
```

---

## 功能概览

- **账号**：云机/容器列表、备注、状态、VPC 等
- **全部对话**：多容器会话聚合、搜索、未读、备注展示
- **聊天**：消息列表、文本/图片收发、引用回复、编辑、删除、表情、长按菜单（回复/转发/复制/收藏/删除）、图片大图预览与转发
- **媒体**：缩略图 + 原图（`/api/v1/media/stream`，见 `ios_app/README.md`）、本地相册选择与多图发送、简单图片编辑（旋转/裁剪/画笔/框选/文字）
- **工具**：话术模板、图片模板，可与聊天收藏联动
- **我的**：个人与设置入口
- **连接**：WebSocket 同步、APNS 推送、未读与预览刷新

## 环境与运行

- **Xcode**：推荐最新稳定版，目标 iOS 14+（部分 UI 需 iOS 16+）
- **依赖**：无第三方包管理器，仅系统框架（SwiftUI、Photos、AVFoundation、UserNotifications 等）
- **配置**：接口地址等由 `APIConfig` 及工程内配置统一管理，按需修改 host / gva 等
- **打开与运行**：用 Xcode 打开 `ios_app/WudiApp.xcodeproj`，选择真机或模拟器运行
